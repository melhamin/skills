# Example Skill

This is a starter skill so the repo installs cleanly from day one. Replace it
with your own.

## How to make your own skill

1. Duplicate this folder under `skills/` and give it a kebab-case name.
2. Update the `name` (must match the folder name, lowercase + hyphens) and
   `description` in the frontmatter above. The `description` is what an agent
   reads to decide when to use the skill, so make it specific.
3. Write the instructions below — this is the content the agent follows.

## When to use

activate this skill.
